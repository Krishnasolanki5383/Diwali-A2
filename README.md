Wix Website Clone – Bump & Beyond

A stylish frontend clone inspired by Wix templates, designed for a blog/community website called “Bump & Beyond”. This project focuses on layout creativity, positioning, and visual storytelling using HTML and CSS.

📌 Project Overview

This project recreates a Wix-style website layout featuring:

Header with responsive preview icons
Hero banner with typography
Navigation menu
Image grid sections
Blog/forum highlights
Footer with subscription form

It is built to practice UI design, positioning, and layout structuring.

🚀 Features
🎨 Wix-style header with device preview icons
🧭 Navigation menu (Home, Forum, Blog, etc.)
